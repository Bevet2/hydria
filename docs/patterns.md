# Patterns

Hydria now maintains a lightweight pattern library in `backend/src/patterns`.

Stored items:
- validated patterns
- reusable templates
- successful strategies

Sources:
- internal task learnings
- selected external patterns from GitHub research

Main modules:
- `pattern.library.js`
- `pattern.search.js`
- `pattern.validation.js`
- `pattern.versioning.js`

The library is used by the strategy layer to bias execution toward proven structures instead of vague reuse.
